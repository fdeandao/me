windows
=======

* n next
* p previous

* c create

* 0-9 go to window #

* , rename


others
======

* t show time
* source-file ~/.tmux.conf - reload config

pane
====

* % vertical split pane
* " horizontal split pane

* ! break pane out of the current window

* o next pane
* ; previous pane

* { swap pane with previous
* } swap pane with next

* x kill current pane
* q show ane indexes

* C-o rotate panes
* M-o rotate panes inverse

* arrows move to pane
* C-arrows resize pane

custom pane
-----------

* h move to left pane
* l move to right pane
* j move to down pane (below)
* k move to up pane (above)

selection
=========

* [ go in selection mode
* ] paste selection
* <space> start selection
* <enter> end selection
* <esc> clear selection
